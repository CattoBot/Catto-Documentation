# `/voice bitrate`
### Descripción
Establece la tasa de bits de tu canal de voz personalizado a la que tú quieras.

### Uso

```
/voice bitrate <bitrate>
```

### Argumentos

`bitrate` - Tasa de bits que deseas aplicar al canal de voz

### Ejemplo

```
/voice bitrate bitrate:64
```
