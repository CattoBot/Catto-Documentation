# `/voice reset`
### Descripción
Reinicia la configuración del canal y restáuralo a la configuración por defecto.

!!!danger ATENCIÓN
Al usar este comando no se muesta un mensaje de confirmación y los cambios efectuados no son reversibles.
!!!

### Uso

```
/voice reset
```
