# `/voice limit`
### Descripción
Establece el límite de usuarios de tu canal de voz en el que tú desees.

### Uso

```
/voice limit <límite>
```

### Argumentos

`límite` - Límite de usuarios para el canal

### Ejemplo

```
/voice limit límite:15
```
