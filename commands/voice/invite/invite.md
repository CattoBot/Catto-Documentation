# `/voice invite`
### Descripción
Envía a un usuario un mensaje de invitación a tu canal de voz personalizado.

### Uso

```
/voice invite <usuario>
```

### Argumentos

`usuario` - Usuario a invitar al canal

### Ejemplo

```
/voice invite usuario:gacarbla
```
