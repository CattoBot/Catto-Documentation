# `/mod vc channel`
### Descripción
Obtén todos los datos posibles de un canal de voz y sus integrantes, y accede a una herramienta única de gestión de éste.

### Uso
```
/mod vc channel <canal>
```

### Argumentos
`canal` - Canal a ser monitorizado

### Ejemplo
```
/mod vc channel canal:#voz 1
```
