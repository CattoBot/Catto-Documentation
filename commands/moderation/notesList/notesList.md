# `/mod notes list`
### Descripción
Obtén una lista con todas las notas que a cierto usuario se le han dado.

### Uso
![](cmd.jpg)
```
/mod notes list <usuario>
```

### Argumentos
`usuario` - Usuario del que se desean obtener las notas

### Ejemplo
```
/mod notes list usuario:gacarbla
```
